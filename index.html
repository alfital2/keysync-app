Features

It just keeps the language right.

macOS has a per-app input setting. It forgets, lags, and resets. FlicKey is what it should have been.

CORE

Per-app language memory

FlicKey remembers the right input language for every app and switches the moment you focus it. Terminal stays Latin, chat stays Hebrew. It's reliable where the built-in macOS setting drops the ball.

Per-site memory

Your browser remembers the language for each website, so the right script is armed before you press a single key.

Fix wrong-layout text

Slipped anyway? One shortcut converts gibberish back to what you meant, between any two layouts, in any app.

EN

Menu-bar indicator

A live language badge sits in your menu bar, so you always know the active layout at a glance, never guessing.

How it works

Up and running in 30 seconds.

1

Download

Grab the 1.0 MB app and drag it to Applications. No account, no installer wizard.

2

Grant Accessibility

Turn on one toggle in System Settings so FlicKey can read and rewrite the text you type.

3

It just works

Type away. Double-tap ⇧ whenever you slip, and let auto-switching handle the rest.

Privacy & security

Private by design.

It's a tiny local utility. Nothing about you ever leaves your Mac.

Stays on your Mac

It only stores tiny preferences - which layout you like per app, website, and chat. No documents, no messages, no keystrokes, no history.

No account, no tracking

No sign-in, no analytics, no telemetry. It never reports what you do - there's no server to report to.

One network call

The only connection it makes is a version check to GitHub - and it sends nothing about you. That's it.

Permissions, explained

Two macOS permissions, each for one clear job. Tap a row to see exactly what it does - and doesn't.
